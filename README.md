# SEGIP — Sistema maestro de registro e identificación personal. Almacena y gestiona los datos biométricos y demográficos de todos los ciudadanos bolivianos. Asigna el CI como identificador único universal. Mantiene relaciones familiares (cónyuge, padre, madre) mediante claves recursivas dentro de su propia tabla. Sirve como fuente autoritativa de identidad para la interoperabilidad entre sistemas estatales.
# Bono Juana Azurduy (BJA) — Sistema transaccional de gestión de transferencias monetarias condicionadas. Registra el ciclo completo de atención materno-infantil: inscripción de la beneficiaria, controles prenatales (4 máximo), parto institucional, y controles pediátricos (12 hasta los 2 años). Cada evento clínico completado dispara automáticamente una orden de pago. Vincula el CI de la madre con el CI del recién nacido en la tabla de partos, estableciendo la trazabilidad madre-hijo post-nacimiento.
# Subsidio Universal Prenatal por la Vida (SUP) — Sistema de gestión logística de entregas nutricionales con mecanismo de voucher. Opera una cadena de tres pasos:
- El control médico genera una boleta numerada. 
- La boleta autoriza el retiro del paquete nutricional. 
- La entrega se registra contra esa boleta consumida. 
- Mantiene un catálogo de paquetes estandarizados por trimestre de gestación y lleva trazabilidad de lotes de producción desde el SEDEM hasta la beneficiaria final.
